## Qué es este Vault

Base de conocimiento interna de **Vesvank** - startup fintech venezolana de infraestructura de pagos. El founder usa este vault para documentar y organizar todo el conocimiento relacionado con la empresa.

## Sobre Vesvank

- **Producto**: Plataforma de pagos que permite a negocios gestionar cobros a través de links de pago, solicitudes de pago y un dashboard centralizado
- **Métodos de pago soportados**: Pago móvil (Bancaribe, Mercantil), Binance Pay (USDT)
- **Funcionalidades clave**: Gestión de clientes, catálogo de productos, tasas de cambio automáticas, facturación digital, orquestación de pagos
- **Mercado**: Negocios venezolanos
- **Web**: vesvank.com
